
#### 回文数

1. 什么是回文数
   1. 对称的数字被成为回文数，比如 `12321` 或者 `2332`


2. 判断回文数的方式一，借助于字符串
```java
public static boolean huiwen(int a) {
    String s = String.valueOf(a);
    // 获取到存放字符串的char数组，这样比较起来方便
    char[] c = s.toCharArray();
    for(int i = 0; i < c.length / 2; i ++) {
        if( c[i] != c[ c.length - 1 - i ] ) return false;
    }
    return true;
}
```


3. 判断回文数的方式二，使用数学公式
```java
public static boolean huiwen(int a) {
    // 每次取最高位和最低位做比较，所以求最高位是很麻烦的
    // 比如: 12321 的最高位要 /10000 才能得到，而 10000 是不确定的，下次可能是1000，所以要动态求
    // 最低位，%10 就可以得到
    // 比较完成后，就需要去掉最高位和最低位，也就是变成 232
    // 然后重复上面的步骤，继续去最高位和最低位做比较，然后在 去掉最高位和最低位 3，此时没有最高位和最低位了
    // 或者最高位和最低位为 22 去掉后就变成了 0，此时也没有最高位和最低位了
    // 所以最后去掉最高位和最低位，后得到的数据小于 10 ，表示循环结束，判断完成
    int nub = a;
    while(nub > 10) {
        // 得到最高位，是10的多少平方，比如 23000 得到的就是 4  10的4次方 --> 10000
        int maxW = (int)Math.log10(nub);
        // 得到最高位
        int max = (int)(nub / Math.pow(10, maxW));
        // 得到最低位
        int min = nub % 10;
        if(max != min) return false;

        // 把最高位和最低位去掉，继续循环 ( 12321 -->  232  去掉了最高位和最低位 )
        nub = (int)(nub % Math.pow(10, maxW) / 10);
    }

    return true;
}
```



4. 判断回文数的方式三，数字逆置，然后进行比较
```java
// 需要的数组和目标值最好动态传入
public static boolean huiwen(int a) {
    // 逆置的方法有很多种，这里借助是StringBuffer的方法
    String b = String.valueOf(a);
    String c = new StringBuffer(b).reverse().toString();
    return b.equals(c);
}
```
