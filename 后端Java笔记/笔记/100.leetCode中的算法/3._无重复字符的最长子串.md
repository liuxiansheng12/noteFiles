

#### 题解
```java
给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
示例 2:

输入: "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
示例 3:

输入: "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```



#### 示范代码1
```java
public static int chong(String str) {
    char[] c = str.toCharArray();

    int chang = 0;
    for(int i = 0; i < c.length; i ++) {
        int ch = 1;
        HashSet<Character> s = new HashSet<Character>();
        s.add(c[i]);
        for(int j = i + 1; j < c.length; j ++) {
            if( s.contains(c[j]) ) break;
            else {
                s.add(c[j]);
                ch ++;
            }
        }
        if(ch > chang) chang = ch;
    }

    return chang;
}
```





#### 示范代码2
```java
public static int chong(String str) {
    char[] c = str.toCharArray();
    HashMap<Character, Integer> s = new HashMap<Character, Integer>();
    int chang = 0;
    int ch = 0;
    for(int i = 0; i < c.length; i ++) {
        // 说明之前存在
        if( s.containsKey( c[i] ) ) {
            // 判断是不是在连续中
            if( i - s.get(c[i]) <= ch ) {
                if (chang < ch) chang = ch;
                // 当前的减去之前的，得到的长度，就是当前前面的查询到当前，并且没有重复的长度，这样就不用回头查了（核心）
                ch = i - s.get(c[i]);
            }
            else {
                ch ++;
            }
        }
        else {
            ch ++;
        }
        s.put(c[i], i);
    }
    if(chang < ch) chang = ch;
    return chang;
}
```
