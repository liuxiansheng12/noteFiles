


#### 题意
1. 给出两个 **非空** 的链表用来表示两个非负的整数。
   1. 其中，它们各自的位数是按照 **逆序** 的方式存储的，并且它们的每个节点只能存储 **一位** 数字。
2. 如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。
3. 您可以假设除了数字 0 之外，这两个数都不会以 0 开头。
4. 示例：
```java
    输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
    输出：7 -> 0 -> 8
    原因：342 + 465 = 807
```




#### 示范代码
```java
private static class Node {
    public int s;
    public Node next = null;
    public Node() {};
    public Node(int n) {
        this.s = n;
    };
}

public static Node sum(int n1, int n2) {
    Node root = null;
    // 构建逆向链表
    Node node1 = null;
    Node node2 = null;
    int n1Len = (int)Math.log10(n1);
    int n2Len = (int)Math.log10(n2);
    while( n1Len >=0 || n2Len >= 0 ) {
        if( n1Len >= 0 ) {
            int n1Max = n1 / (int)Math.pow(10, n1Len);
            Node node = new Node(n1Max);
            node.next = node1;
            node1 = node;
            n1 = n1 - n1Max * (int)Math.pow(10, n1Len);
            n1Len --;
        }
        if( n2Len >= 0 ) {
            int n2Max = n2 / (int)Math.pow(10, n2Len);
            Node node = new Node(n2Max);
            node.next = node2;
            node2 = node;
            n2 = n2 - n2Max * (int)Math.pow(10, n2Len);
            n2Len --;
        }
    }
    // 进行相加
    int jin = 0;
    Node nodes = null;
    while(node1 != null || node2 != null || jin > 0) {
        int a = node1 != null ? node1.s : 0;
        int b = node2 != null ? node2.s : 0;
        int sum = a + b + jin;
        int n = sum % 10;
        // 创建节点
        Node node = new Node(n);
        if(root == null) root = node;
        else {
            nodes.next = node;
        }
        nodes = node;

        if(node1 != null) node1 = node1.next;
        if(node2 != null) node2 = node2.next;

        jin = sum / 10;
    }


    // 打印查看结果
    Node roots = root;
    while(roots != null) {
        System.out.println(roots.s);
        roots = roots.next;
    }

    return root;
}
```
