2. 示范代码
```java
public static int kai(int n) {
    // 平方根一定在 0 - n 之间，取中间数，一个加，或者一个减，总会重合
    
    int zhong = 0;
    int start = 0;
    int end = n;

    while( start <= end ) {
        zhong = (end - start) / 2 + start;
        int ping = zhong * zhong;
        if( ping == n) return zhong;
        else if( ping < n ) start = zhong + 1;
        else if( ping > n ) end = zhong - 1;
    }
    System.out.println(start);
    return -1;
}
```
