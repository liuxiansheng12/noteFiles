- 获取列表中的某一位
```py
arr = [1, 2, 3, 4, 5]
print(arr[2]) # 打印 3
```

- 获取列表中的某一段，和字符串使用方式也原理一样
```py
arr = [1, 2, 3, 4, 5]
print(arr[2:5]) # 打印 [3, 4, 5]
print(arr[2:5:2]) # 打印 [3, 5]
```


- 往列表的最后添加数据
```py
arr = [1, 2, 3, 4, 5]
arr.append(7)
print( arr ) # 打印 [1, 2, 3, 4, 5, 7]
```

- 在列表的中间某一位插入数据
```py
arr = [1, 2, 3, 4, 5]
arr.insert(3, 9)
print( arr ) # 打印 [1, 2, 3, 9, 4, 5]
```
